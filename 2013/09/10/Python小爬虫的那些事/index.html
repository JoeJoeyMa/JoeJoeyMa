Python

从接触计算机以来，一直都有着想深入了解一门语言。也是因为有做着网络电商的原因吧，店家每天都一直维护干着同样或者是类似流水线式的工作,每天得上传自己的产品修改着宝贝详细里的描述，有时也得观察着同行的一举一动。让我拥有了想做一个真正脚本的想法念头，解放自己的劳动力！
俗话说的好解放劳动力才能增大自己的生产里，从琐碎事里摆脱，做着认为对自己有用的事情才能增长。
